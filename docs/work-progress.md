- UI 문구에서 "실시간 공유" 표현을 제거했습니다.
- 방 관련 패널 문구를 간결화했습니다.
  - "실시간 공유를 위해 ..." → "Supabase 환경변수를 설정해주세요."
  - "실시간 방 만들기" → "방 만들기"
  - "실시간 방 연결됨" → "방 연결됨"
- 사내 레지스트리 403 대응 문서를 추가했습니다.

## 주의사항
- 현재 UI에서는 "실시간" 단어를 제거했지만, 기능 자체는 room 기반 동기화를 사용합니다.
- 사내 레지스트리 정책이 해결되지 않으면 신규 패키지 설치가 계속 실패할 수 있습니다.
- 인증 토큰은 저장소에 커밋하면 안 됩니다.

## 사용자가 해야 할 일
- 사내 npm 레지스트리 인증 재설정 (`pnpm login` 또는 `.npmrc` 토큰)
- 필요 시 플랫폼/보안팀에 패키지 미러링 요청
- 환경변수 설정
  - `NEXT_PUBLIC_SUPABASE_URL`
  - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
